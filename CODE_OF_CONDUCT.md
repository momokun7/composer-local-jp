# 行動規範

## 私たちの約束

このプロジェクトでは、すべての参加者にとってオープンで友好的な環境を提供することを目指しています。

このプロジェクトは [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) に基づいています。

## 基本ルール

- **敬意を持ったコミュニケーション**: 他の参加者に対して礼儀正しく、建設的な態度で接してください
- **多様性の尊重**: 経験、技術レベル、バックグラウンドに関わらず、すべての人を歓迎します
- **建設的なフィードバック**: 批判は個人ではなくアイデアに対して行い、改善提案を添えてください
- **責任ある行動**: 自分の言動がコミュニティに与える影響を考慮してください

## 許容されない行為

- ハラスメント、侮辱、差別的な発言
- 個人攻撃や政治的な挑発
- 他者のプライバシー情報の無断公開
- その他、プロフェッショナルとして不適切と判断される行為

## 対応

行動規範に違反する行為を目撃した場合は、Issue でプロジェクトメンテナーに報告してください。すべての報告は公平かつ適切に対応します。

## 帰属

この行動規範は [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) を元に作成されています。
